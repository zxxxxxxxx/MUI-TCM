## 说几句：
当初查PWA时建议使用pouchdb，可以做到离线访问，因为没时间做后端刚好IBM有个云数据库可以用，果断上手。一用发现，只有英文文档，而且IBM访问速度略慢，好像国内对PWA也并不是很感兴趣，所以劝退。
